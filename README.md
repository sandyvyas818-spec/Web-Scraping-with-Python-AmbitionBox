# 🌐 Web Scraping with Python – AmbitionBox

## 📌 Project Overview

This project demonstrates web scraping using Python to collect company information from AmbitionBox.

The project extracts useful information such as:

- Company Name
- Ratings
- Reviews
- Salaries
- Interviews
- Jobs
- Benefits
- Photos

The collected data is cleaned and stored in a Pandas DataFrame for further analysis.

## 📊 Workflow

1. Send HTTP Request

5. Clean Data
6. Export CSV
